# Portfolio Website

This repository contains the source code for my personal portfolio website.

The site serves as a central place to document and present my work across multiple disciplines, including software development, writing, and digital art. It is designed to function both as a portfolio and as a personal archive of projects, experiments, and ideas.


## Overview

This website brings together several areas of my work:

- **Software Development**
  - Cybersecurity
  - Web development
  - Game development
  - AI / Machine Learning
  - Data Science and analytics
  - Experimental tools and systems

- **Writing**
  - Online published books
  - Upcoming physical publications
  - Essays and short-form writing

- **Digital Art**
  - Character art
  - Concept sketches
  - Personal creative work

The blog functions as a lightweight knowledge log where I post thoughts, technical notes, development logs, and other small pieces of writing.


## Philosophy

Rather than using a traditional CMS, the site uses a **Git-based content workflow**.  
All content is written in Markdown/MDX and stored directly in the repository.

This approach provides:

- version control for all writing and posts
- simple long-term maintenance
- full ownership of content
- a lightweight publishing workflow

New posts can be scaffolded automatically using a helper script.



## Tech Stack

The site is built using modern web technologies.

**Framework**
- Next.js (App Router)

**Language**
- TypeScript

**Styling**
- Tailwind CSS

**Content**
- Markdown / MDX

**Content Parsing**
- gray-matter
- remark / rehype

**Automation**
- Python scripts for blog post scaffolding
